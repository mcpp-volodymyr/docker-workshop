# Docker Workshop

Hands-on Docker workshop with practical examples, demos, and exercises.

---

## 📦 Setup

Install Docker:

https://docs.docker.com/get-docker/

Verify installation:

```bash
docker run hello-world
```

---

## 📚 Structure

- `session-1/` — Docker basics + demo + homework  
- `session-2/` — Practical Docker (solutions, debugging)  
- `session-3/` — CI/CD examples  

---

## 🚀 Getting Started

Go to:

```bash
cd session-1
```

Then follow the instructions in:

```
homework.md
```

---

## 💡 Notes

- All commands are copy-paste ready  
- You don’t need to memorize anything  

---

## 🎯 Goal

By the end of Session 1, you should be able to:
- build a Docker image  
- run a container  
- access a service in your browser
